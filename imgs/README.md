# Screenshots

This folder contains screenshots for the README.

## Required Screenshots:

1. **main-menu.png** - Screenshot of the main menu showing the ASCII art banner and menu options
2. **system-info.png** - Screenshot showing system information output in JSON format
3. **network-scanner.png** - Screenshot of the network scanner in action
4. **port-scanner.png** - Screenshot of the targeted IP/Port scanner

## How to Create Screenshots:

1. Run the application: `cargo run`
2. Take screenshots of each menu option in action
3. Save them in this folder with the names listed above
4. Recommended size: 800-1200px width for clarity

## Tips for Good Screenshots:

- Use a clean terminal window
- Make sure colors are visible
- Capture the full output
- Use Windows Snipping Tool or similar
